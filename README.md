# puzzle-game

브라우저에서 실행되는 과일 짝맞추기 게임입니다.  
난이도를 `쉬움 / 보통 / 어려움`으로 고를 수 있고, 각 난이도마다 보드 크기와 제한시간이 달라집니다.

## 게임 규칙

- 같은 과일 카드 2장을 찾아 모든 카드를 맞추는 게임입니다.
- 첫 카드를 누르는 순간 제한시간 카운트다운이 시작됩니다.
- 시간이 0초가 되면 게임이 즉시 종료됩니다.
- 틀린 카드 2장을 고르면 잠시 보여준 뒤 다시 닫힙니다.
- 남은 시간이 30초 이하가 되면 경고 상태, 10초 이하가 되면 위험 상태로 표시됩니다.

## 난이도

| 난이도 | 보드 크기 | 제한시간 |
| --- | --- | --- |
| 쉬움 | 4x4 | 4분 |
| 보통 | 6x6 | 2분 |
| 어려움 | 8x8 | 2분 |

## 코드 구조

### `index.html`

게임 화면의 기본 마크업입니다.  
난이도 버튼, 상태판, 메시지 영역, 보드 컨테이너를 포함합니다.

### `js/utils.js`

공통 설정과 유틸 함수를 관리합니다.

- 난이도별 설정
- 과일 목록
- 카드 셔플
- 시간 포맷
- 정확도 포맷

### `js/game.js`

게임의 핵심 로직을 담당합니다.

- 난이도 전환
- 보드 크기 변경
- 덱 생성
- 카드 클릭 처리
- 매칭 판정
- 제한시간 카운트다운
- 시간 초과 처리
- 게임 완료 처리

### `js/board.js`

카드 배열 상태를 실제 DOM 버튼으로 렌더링합니다.

### `js/app.js`

필요한 DOM 요소를 찾아 `MemoryPuzzleGame` 인스턴스를 생성합니다.

### `css/styles.css`

전체 레이아웃과 시각 스타일을 담당합니다.

- 좌우 패널 레이아웃
- 난이도 선택 UI
- 상태 카드
- 보드 그리드
- 카드 뒤집기 애니메이션
- 타이머 경고/위험 스타일

## 실행 방법

별도 빌드 없이 바로 실행할 수 있습니다.

1. 프로젝트 폴더를 엽니다.
2. `index.html`을 브라우저에서 실행합니다.
